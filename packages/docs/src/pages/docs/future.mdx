# Future plans

BlinkDB is not yet mature. While BlinkDB *should* reliably work for every use case
documented in these docs (please file a bug ticket if it doesn't), **it is not
yet as fast as it could be**, and **some features are not yet implemented**.

But don't worry - BlinkDB will only improve over time. If you aren't convinced with the
current feature set, check back when **1.0.0** is released!
