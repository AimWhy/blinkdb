# remove()

Removes a given entity from the table.

```ts
// Remove Alice from the table
await remove(userTable, { id: "alice-uuid" });
```

| Parameter | Description                                                              |
| --------- | ------------------------------------------------------------------------ |
| `table`   | The table created by [`createTable()`](/docs/reference/createTable) .    |
| `item`    | The item (or primary keys of an item) which gets removed from the table. |
