# first()

Retrieves the first entity from the table.

If a [filter](/docs/filters) is supplied, "first" means the first item that would be returned
from a similar call to [`many()`](/docs/reference/many) .

```ts
const oldestUser = await first(userTable, {
  sort: {
    key: "age",
    order: "desc",
  },
});
```

| Parameter | Description                                                                                                    |
| --------- | -------------------------------------------------------------------------------------------------------------- |
| `table`   | The table created by [`createTable()`](/docs/reference/createTable) .                                          |
| `filter`  | Optional. If given, the item returned will be the first one to match the filter. See [filters](/docs/filters). |
